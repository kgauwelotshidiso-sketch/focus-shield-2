# Focus Shield Dashboard

Purpose:
Provide users with a quick overview of their progress and motivation.

Widgets:

- Current Streak
- Longest Streak
- Today's Affirmation
- Goals Shortcut
- Focus Timer Shortcut
- Concentration Shortcut
- Progress Shortcut

Example Affirmation:

"I pause, I listen, and I follow my dreams."

Future Features:

- Recovery Statistics
- Productivity Metrics
- Daily Challenges
