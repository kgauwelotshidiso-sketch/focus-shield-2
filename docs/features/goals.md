# Goals System

Purpose:
Help users remember what they are working toward.

Features:

- Create Goals
- Edit Goals
- Delete Goals
- Goal Categories
- Daily Goal Review
- Long-Term Vision

Examples:

Career:
Become an Electrical Engineer.

Personal:
Master listening without interrupting.

Health:
Become fit and disciplined.

Dashboard Integration:

- Show top goals.
- Show daily reminders.
- Show progress indicators.
