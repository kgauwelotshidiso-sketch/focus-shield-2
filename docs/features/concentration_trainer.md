# Concentration Trainer

Purpose:
Train the user's ability to hold attention on a single thought.

Features:

- Select affirmation
- 1 minute session
- 2 minute session
- 5 minute session
- 10 minute session
- Session tracking
- Daily streak

Example:

"I pause, I listen, and I follow my dreams."

Statistics:

- Sessions completed
- Total concentration time
- Weekly consistency

Future Features:

- Guided concentration
- Audio mode
- Progress graphs
